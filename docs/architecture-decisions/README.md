# Architecture Decision Records

This folder contains Architecture Decision Records for my Cloud and Platform Engineering projects.

An Architecture Decision Record explains:

- The problem or decision being considered
- The available options
- The selected approach
- The reasons for the selection
- The trade-offs and consequences
- Future review conditions
